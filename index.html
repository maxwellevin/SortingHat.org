Sorting Hat

This program assigns students to class sections based on their section preferences while maintaining administrative requirements for legal sex or athlete balance. Please read the docs to learn how to format the section and student input files and to learn more about SortingHat.

  1. Choose Sections File

  2. Choose Students File

  3. Set Minimum Sex Ratios
    0%
    0%

    These sliders set the minimum percentage of male or female students allowed in any given section/class. Setting both sliders to 0% means that sex should not be a consideration for course allocations.

  4. Set Athlete Ratio 100%

    These sliders set the maximum percentage of student-athletes allowed in any given section/class. Setting this to 100% means that student-athlete proportion should not be a consideration for course allocations.

  5. Run the Program
  6. Save the Results to File
